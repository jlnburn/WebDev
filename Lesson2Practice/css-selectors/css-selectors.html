Types of Selectors

Type Description Example
Element <p>: The Paragraph element. The <p> HTML element represents a paragraph. p { margin: 0; text-indent: 3ch; }
Class .class: The class attribute specifies one or more classnames for an element. <h2 class="country">USA</h2>
.country { background-color: black;}
ID #id: The id selector is used to select one unique element. <p id = "para">Center text in this paragraph>
#para {text-align: center;}
Universal The universal selector selects all HTML elements on the page. * {text-align: center;}
Pseudo-class :hover: A pseudo-class is used to define a special state of an element. :hover {background-color: blue;}
Pseudo-element ::before: A CSS pseudo-element is used to style specified parts of an element. h1::before {content: url(smiley.gif);}
Combinator .parent > .child: A combinator explains the relationship between the selectors div > p {background-color: yellow;}
Attribute [href$=".pdf"]: An attribute selector is used to select elements with a specified attribute. a[target] {background-color: yellow;}
