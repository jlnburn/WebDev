Lesson 3

Terminology

Flex Container
A flex container is an HTML element whose display property's value is flex or inline-flex.
Flex Items
Flex items are the direct children of a flex container.
Flex Directions
The flex direction defines the direction flex items are placed in the flex container.
Main axis
The defining direction of how the flex items are placed in the flex container.
Main size properties
The main size of a flex item is the size it has in the main dimension.
Cross axis
The cross axis in flexbox runs perpendicular to the main axis.
Cross size properties
The cross size property is whichever of ‘width’ or ‘height’ that is in the cross dimension.

Properties

Flex Container Properties

display
display:flex;/* enables flex context for direct children. */
flex-direction
flex-direction: row;  /* establishes the main-axis. */
flex-wrap
flex-wrap: nowrap;  /* all flex items will be on one line */
flex-flow
flex-flow: column wrap;  /* flex-direction and flex-wrap properties. */
justify-content
justify-content:flex-start;/* items packed to start of the flex-direction.*/
align-items
align-items: stretch;/* fill the container respecting min-widthmax-width. */
align-content
align-content: flex-start;  /* items packed to the start of the container.*/

Flex Item Properties

flex-grow
flex-grow: 3;/* represents a growth factor of 3.*/
flex-order
order; -3;/* moves items 3 places column or row.*/
flex-shrink
flex-shrink: 2;/* represents a shrinkage factor of 2.*/
flex-basis
flex-basis: 200px;/* the flex item starts at 200px.*/
flex
flex: 0 0 200px;/* item does not grow or shrink and starts out at 200px. */
align-self
align-self: center;/* align items to the middle of the container. */
