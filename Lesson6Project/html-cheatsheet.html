HTML Tags And Attributes

Head Group

head tag

head /head

Defines information about the document.

title tag

title /title

Defines the title of a document.

link tag

link /link

Defines the relationship between a document and an external resource.

meta tag

meta /meta

Defines metadata about an HTML document.

style tag

style /style

Defines style information for a document.

Sectioning Group

article tag

article /article

Article specifies independent, self-contained content.

aside tag

aside /aside

Defines content aside from the content it is placed in.

main tag

main /main

Defines the main content of a document.

div tag

div /div

Creates containers where ever they are needed.

fieldset tag

fieldset /fieldset

Represents a set of form controls optionally grouped under a common name.

Grouping Group

p Paragraph tag

p /p

Groups text into related sections.

ol ordered list tag

ol li /ol li

An intentionally ordered list.

ul unordered list tag

ul li /ul li

Order of the list is not important.

dl descriptive list tag

dl dt /dl dd

Lists descriptive term and definition.

Text Group

p Paragraph tag

p /p

Groups text into related sections.

span tag

span /span

Inline tag that can be used to create a section in a paragraph.

br Break tag

br /br

This tag represents a line break.

h heading tag

h1 /h1, h2 /h2, h3 /h3, h4 /h4, h5 /h5, h6 /h6

Headings, getting smaller font from top to bottom.

strong tag

strong /strong

A semantic hin that marks the text inside it as strong.

Table Group

Table tag

table /table

Creates a table

tr tag

tr /tr /*rowspan, align, colspan, nowrap*/

Sets off each row in a table.

td tag

td /td /*rowspan, align, colspan, nowrap*/

Sets off each cell in a row.

th tag

th /th

Sets off the table header (a normal cell with bold, centered text.

Form Group

form *Global tag

form /form

Defines a form.

Action tag

action; /*action_page.php*/

Defines the action to be performed when the form is submitted.

Target tag

target; /*_blank, _self, _parent, _top */

Specifies where to display the response that is received.

method tag

method; /*method="get", method="post"*/

Specifies the HTTP method to be used when submitting the form.

autocomplete tag

autocomplete

Specifies whether a form should have autocomplete on or off.

novalidate tag

novalidate

Specifies that the form-data (input) should not be validated when submitted.
