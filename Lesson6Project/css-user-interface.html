CSS User Interface

Box-Shadow

none:

No shadow is given.

/*box-shadow: none;*/

h-offset/v-offset

The horizontal and vertical offset of the shadow.

/*box-shadow: 5px 10px 10px 10px;*/

blur

The blur radius.

/*box-shadow: 20px 20px 10px grey;*/

spread

The spread radius.

/*box-shadow: 20px 20px 50px 15px grey;*/

color

The color of the shadow.

/*box-shadow: 20px 20px 20px 10px red;*/

inset

Changes the shadow from an outer shadow to an inner shadow

/*box-shadow: 20px 20px 50px 10px pink inset;*/

Radius

length

Defines the shape of the corners.

/*border-radius: 100px;*/

%

Defines the shape of the corners in %

/*border-radius: 10%;*/

initial

Sets this property to its default value.

/*color: initial;*/

inherit

Inherits this property from its parent element.

/*color: inherit;*/

Colors

color

Specifies the text color.

/*color: red;*/

initial

Sets this property to its default value.

/*color: initial;*/

inherit

Inherits this property from its parent element.

/*color: inherit;*/

Fonts

font-style

Specifies the font style.

/*font-style: normal;*/

font-variant

Specifies the font variant.

/*font-variant: normal;*/

font-weight

Specifies the font weight.

/*font-weight: bold;*/

font-size/line-height

Specifies the font size and the line-height.

/*font-size: x-small;*/

font-family

Specifies the font family.

/*font-family: Georgia;*/

Lists

list-style

Sets all the properties for a list in one declaration

/*list-style-type: none;*/

list-style-image

Specifies an image as the list-item marker

/*list-style-image: url('sqpurple.gif');*/

list-style-position

Specifies the position of the list-item markers.

/*list-style-position: outside;*/

list-style-type

Specifies the type of list-item marker

/*ul.a {list-style-type: circle;}*/

Background

background-color

Specifies the background color to be used

/*background-color: red;*/

background-image

Specifies background images to be used.

/*background-image: url('paper.gif');*/

background-position

Specifies the position of the background images

/*background-position: left top;*/

background-size

Specifies the size of the background images

/*background-size: auto;*/

background-repeat

Specifies how to repeat the background images

/*background-repeat: repeat;*/

background-origin

Specifies the positioning area of the background images

/*background-origin: padding-box;*/

background-clip

Specifies the painting area of the background images

/*background-clip: border-box;*/

background-attachment

Specifies whether the background images are fixed.

/* background-attachment: fixed;*/
