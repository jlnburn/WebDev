Terminology

Animations
Animations let us decide if and how an animation repeats, give us granular control over what happens throughout the animation, and more.
Keyframes
Keyframes CSS at-rule controls the intermediate steps in a CSS animation sequence by defining styles for waypoints along the animation sequence.
Transforms
Transforms let us rotate, scale, skew, or translate an element. It modifies the coordinate space of the CSS visual formatting model.
Transitions
Transitions allow us to animate CSS properties from an original value to a new value over time when a property value changes.
Easing
Easing accounts for the rate at which a transition occurs.
Rotation
A rotation function causes an element to be rotated around an axis, or around an arbitrary vector in 3D space.
Scale
The scale function defines a transformation that resizes an element on the 2D plane.
Skew
The skew function defines a transformation that skews an element on the 2D plane.
Delay
The transition-delay property specifies the duration to wait before starting a property's transition effect when its value changes.
Transition Timing
The transition-timing-function property sets how intermediate values are calculated for CSS properties being affected by a transition.

Transform Properties

transform: rotate(20deg)

The transform property applies a 2D or 3D transformation to an element. This property allows you to rotate, scale, move, skew, etc., elements.

div.a {
width: 400px;
height: 200px;
transform: rotate(20deg);
box-shadow: 0 0 5px 5px var(--main-color-200);
background-color: var(--main-color-200);
color: var(--main-color-600);
border: dashed 1px var(--main-color-600);
}

Transform Properties

transform: skewY(20deg)

The skewY() method skews an element along the Y-axis by the given angle.

div.b {

background-color: var(--main-color-200);
color: var(--main-color-600);

}

Transform Properties

transform: translate(30px,20px)

The translate() method moves an element from its current position (according to the parameters given for the X-axis and the Y-axis).

background-color: var(--main-color-200);

box-shadow: 0 0 5px 5px var(--main-color-200);
}

Animation Properties

@keyframes colors

When CSS style is specified inside the @keyframes rule, the animation will gradually change from the current style to the new style at certain times.

.divaa1 {
width: 300px;
height: 200px;
background-color: var(--main-color-400);
animation-name: colors;
animation-duration: 10s;
}
@keyframes colors {
from {background-color: var(--main-color-400);}
to {background-color: var(--main-color-100);}
}

Animation Properties

animation-name: eastbound

The animation-name property specifies a name for the @keyframes animation.

.divaa2 {
width: 300px;
height: 200px;
background-color: var(--main-color-400);
animation-name: eastbound;
animation: eastbound infinite;
animation-duration: 15s;
}
@keyframes eastbound {
from {left: 0px;}
to {left: 200px;}
}

Animation Properties

animation-direction: alternate

The animation-direction property defines whether an animation should be played forwards, backwards or in alternate cycles.

.divaa6 {
width: 300px;
height: 200px;
background-color: var(--main-color-400);
animation: eswenoso 5s 2;
animation-direction: alternate;
}
@keyframes eswenoso {
0% {background: red; left: 0px; top: 0px;}
25% {background: yellow; left: 200px; top: 0px;}
50% {background: blue; left: 200px; top: 200px;}
75% {background: green; left: 0px; top: 200px;}
100% {background: red; left: 0px; top: 0px;}
}

Animation Properties

animation-fill-mode: forwards

The animation-fill-mode property specifies a style for the element when the animation is not playing (before it starts, after it ends, or both).

.divaa5 {
width: 300px;
height: 200px;
background-color: var(--main-color-400);
animation: southboundfill 10s;
animation-fill-mode: forwards;
}
@keyframes southboundfill {
from {top: 0px;}
to {top: 200px; background-color: purple;}
}
