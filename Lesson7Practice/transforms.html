Text Transformation

To rotate a piece of text, you can use the transform property. Rotations go clockwise, so to turn the text anti-clockwise you need to use a negative degree value. The origin helps to set the point where the rotation occurs. The translate property value allows you to move the text into position.

HTML Code

<h2 class="rotate-left">Text Transformation</h2>

CSS Code


.rotate-left {
    display: grid;
    grid-column: 1;
    font-size: 2.5rem;
    width: max-content;
    text-align: right;
    writing-mode: vertical-rl;
    transform-origin: top left;
    transform: rotateZ(180deg);
    transform: translateX(-3rem);
    transform: translateY(-100%);
    border-left: solid 4px var(--accent-color-600); 
}

        border: solid lightgray 1px;
