Terminology

Grid Container
The grid container holds all the contents placed on the grid
Grid Item
The contents of the grid container are called the grid items.
Track
The track runs from one end of the container to the other.
Cell
The cell are squares or rectangles formed by the rows and columns.
Area
A rectangle or square of two or more grid cells.
Explicit Grid
The template columns, rows and area form the explicit grid.
Implicit Grid
Tracks automatically generated when the grid container adds grid lines.
Grid Template
The grid template defines the grid columns, rows and areas.
Gutters
The spaces between grid tracks are called Gutters.
Grid Naming
When grid lines are assigned names that define a grid area.
